Flex-flow é uma abreviação das duas propriedades, direction e wrap.

A propriedade Flex-flow é um atalho que combina as propriedades flex-direction e flex-wrap.

Flex-Direction = Row (Padrão)

Alinha os itens flexíveis em uma linha, da esquerda para a direita (direção padrão)

Flex-Wrap = nowrap (padrão), wrap e wrap-reverse.

Flex-wrap controla a quebra de linha dos itens flexíveis dentro do contêiner. A quebra de linha será feita caso o container ou a tela não deem espaço suficiente.
