Flex Item - Order e Align Self

Order

Com a propriedade order, podemos alterar a posição de um item flex dentro de uma lista de itens flex. Isso afeta apenas a ordem visual, mantendo a estrutura e semântica HTML inalteradas. O valor padrão é order: 0, e itens com valores de order mais altos aparecem mais à direita, enquanto valores negativos os movem para a esquerda.

Align Self

A propriedade align-self permite especificar o alinhamento de um item flex individual dentro do flex container, sobrescrevendo o alinhamento definido pelo container. Este alinhamento é baseado no eixo transversal.
