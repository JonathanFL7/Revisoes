::before + ::after

Tanto before quanto after é de extrema importancia nos projetos, usando esses Pseudo Elementos podemos inserir um novo elemento no conteudo que quisermos no CSS.
Podemos por esses conteudos no HTML, mas teriamos que fazer varias vezes, causando ineficiencia.
